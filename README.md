# Super Curve Finance Portal

オプション実務辞典、GEXダッシュボード、為替介入モデル、金利・原油・海運分析、市場レジーム判定などを統合した金融リサーチポータル。

## Modules

- Option Handbook
- GEX Dashboard
- Oil Regime
- USDJPY Intervention
- Rates Dashboard
- Shipping Dashboard
- Gold / Silver Dashboard
- Macro Library
- Research Notes
- AI Search

## GitHub Pages

`Settings → Pages → Deploy from a branch → main / root` を選択すると公開できます。
