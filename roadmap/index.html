DEVELOPMENT ROADMAP

Roadmap

SUPER CURVE TERMINAL 開発ロードマップ

Phase 1からPhase 5まで、静的ポータルからRAG / Vector DB / API連携まで段階的に拡張します。

Phase 1：Static Portal

Home、Handbook、Dashboard、Library、AI Searchの基本導線を整備する段階です。

GitHub Pages上で、CSS適用・404解消・主要ページ表示を安定化します。

Phase 2：Handbook / Concept Pages

GEX、Gamma Flip、Dealer Long Gamma、Vanna、Charm、0DTEなどの概念ページを整備します。

Formula、Market Impact、Dealer Action、Trader Watchを統一フォーマットで表示します。

Phase 3：Knowledge Graph / Library

概念同士の関係をJSON Knowledge Graphとして整理し、カード型ビューアで閲覧できるようにします。

LibraryからJSONを直接見せず、HTMLビューで知識構造を確認できる構成にします。

Phase 4：AI Search / RAG Ready

検索フォーム、検索ボタン、Enter検索、結果カード表示を整備します。

将来的にはEmbedding / Vector DB / OpenAI APIへ接続し、RAG回答生成へ拡張します。

Phase 5：Live Data / Automation

市場データ、公開API、日次更新スクリプト、ダッシュボード自動生成へ拡張します。

最終的には、GEX、FX、Oil、Rates、Shippingなどのレジーム監視を自動更新する構成を目指します。
